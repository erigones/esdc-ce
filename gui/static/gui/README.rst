Instalation for scss development
================================

In Fedora Linux:

        sudo dnf install rubygems ruby-devel
        sudo gem update --system
        sudo gem install compass
        cd  $ERIGONES_HOME/gui/static/gui
        compass watch

In Ubuntu Linux:
        sudo apt-get install ruby-compass
        cd  $ERIGONES_HOME/gui/static/gui
        compass watch
